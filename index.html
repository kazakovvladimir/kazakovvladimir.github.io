Услуги программиста
Конструирование / прототипирование
  • конструирование веб-сайтов высочайшего уровня на заказ. На выходе получается готовый макет сата ( изображения или HTML ), который можно отдавать дизайнерам.
Верстка одностраничного Landing Page ( PSD to HTML)
  • верстка целевых страниц по готовому дизайну (PSD).
Верстка сложного многостраничного сайта
  • верстка сложных сайтов по готовому дизайну (PSD) и контенту по прототипу и четкому Техническому Заданию. Сайт отвечает всем современным стандартам качества.
PHP / MySQL программирование
  • разработка больших проектов или простых скриптов, всевозможных модулей для вашего сайта.
Адаптация сайтов под мобильные устройства
  • верстка на Bootstrap 3/4 одинаково хорошо смотрится и на компьютерах и на мобильных устройствах.
Красивые эффекты на страницах
  • эффектное появление текстовых или графических блоков в готовом сайте. Используется jQuery, HTML5 и CSS3.
Услуги дизайнера
Создание целого дизайна или отдельных элементов для сайтов
  • создание уникального элегантного дизайна сайта в графических программах по прототипу или ТЗ.
Создание логотипов
  • уникальный логотип для вашей фирмы или сайта.
Работа в графических редакторах
  • всевозможное редактирование изображений.
  • цветокоррекция фотографий.
  • ретушь и восстановление фотографий.
  • создание коллажей.
  • вырезка объектов.
  • добавлениее водяных знаков на фотографии.
  • оптимизация графики для более быстрой загрузки сайта и улучшения показателей Gogle PageSpeed Insights.
  • изменения размеров и форматов изображений для интернет-магазинов.
  • создание Экшенов на заказ.
Конвертация растровой графики в векторную
  • преобразование растравых изображений ( обычно это логотипы ) в векторный формат, размеры которых можно изменять без потерь в качестве.
Склейка и обработка панорам ( PTGui, Photoshop )
  • клею панорамы любой сложности, даже если в автоматическом режиме программы не распознают точки пересечений отдельных кадров.
